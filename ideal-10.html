Ideal 10

Información Digital Estratégica en Línea

¿Quiénes somos?

El diseño y desarrollo del software IDEAL.10 es el resultado del trabajo de un grupo de profesionales interdisciplinarios que se planteó como misión construir una herramienta administrativa financiera y de planeación, que responda a las necesidades en el proceso de información de las entidades públicas, aplicando los marcos normativos para los procesos y subprocesos del modelo administrativo adoptado por los entes territoriales, permitiendo registrar documentos de control administrativo, clasificando y registrando automáticamente documentos de control contable, haciendo uso además de las herramientas de planeación y de ejecución del presupuesto público, obteniendo efectividad en los insumos o resultado de los procesos.

El desarrollo del proyecto es tipo web con funcionamiento en la nube, el lenguaje utilizado para trabajar el Backend es PHP, para Frontend se utiliza JavaScript con el Framework Vue js para la creación de interfaces de usuario, las actualizaciones y control de versiones del software se utiliza git, con el servicio de alojamiento basado en web Bitbucket.


Al sistema aplica la teoría de proceso integrando la información de la entidad pública territorial y contiene los siguientes componentes agrupando módulos:

  • Componente financiero: Presupuesto, Contabilidad.
  • Componente administrativo: Tesorería, Gestión Humana, Control de activos, Almacén.
  • Componente planeación: Planeación estratégica.
  • Componente contractual: Contratación.
  • Componente Apoyo a la administración: Herramientas de MIPG, Administración del software, Módulo de informes, Módulo de parametrización.
