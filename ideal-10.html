IDEAL.10

¿Qué es IDEAL.10?

IDEAL.10 es un software administrado por un grupo de profesionales interdisciplinarios con el fin de tener una herramienta administrativa y financiera actualizada a las necesidades modernas para uso en las entidades públicas, con capacidad de responder a las necesidades actuales en el proceso de información y envío de informes, con la aplicación del régimen contable público en convergencia con NIIF – NICSP de la CGN, el régimen de contabilidad presupuestal pública de la CGR y la aplicación de los nuevos catálogos presupuestales en la etapa de programación y ejecución del ingreso y gasto público, dando cumplimiento a la ley orgánica de presupuesto.

Funcionamiento en la nube.
El desarrollo del proyecto es tipo web con funcionamiento en la nube.
Arquitectura Web: Backend en PHP y Frontend con Vue.js.
El lenguaje utilizado para trabajar el Backend es PHP, para Frontend se utiliza JavaScript con el Framework Vue js para la creación de interfaces de usuario.
Control de Versiones con Git y Bitbucket.
Actualizaciones y control de versiones del software se utiliza git, con el servicio de alojamiento basado en web Bitbucket.
